# Renmay — Quiz (ڕێنمایی قوتابخانە ناحکومییەکان)

سیستەمێکی تاقیکردنەوەی خۆکار (Next.js) بۆ پرسیارەکانی ڕێنمایی قوتابخانە و
پەیمانگە ناحکومییەکان.

## تایبەتمەندییەکان

- هەڵبژاردنی ماددە/بەش پێش دەستپێکردن (یان هەموو پرسیارەکان پێکەوە).
- هەر پرسیارێک بە شێوەی چوار هەڵبژاردەیی (multiple choice).
- پیشاندانی وەڵامی ڕاست/هەڵە دەستبەجێ دوای هەڵبژاردن.
- لاپەڕەی ئەنجام: نمرە بە ڕێژە، ژمارەی ڕاست و هەڵە، و پێداچوونەوەی هەموو
  پرسیارەکان لەگەڵ سەرچاوە (ماددە/لاپەڕەی PDF).
- ڕووکارێکی نوێ، ڕاست-بۆ-چەپ (RTL)، فۆنتی Calibri.
- هیچ داتایەک هەڵناگیرێت — هەموو حیسابکاری لای بەکارهێنەر (client-side) دەکرێت.

## جێبەجێکردن

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # بنیاتنانی بەرهەمهێنان
```

## داتا

پرسیارەکان لە `data/questions.json`ـدان (٣٨٣ پرسیار)، دروستکراو لە فایلی Excel/CSV.
هەر تۆمارێک: پرسیار، وەڵامی ڕاست، و سەرچاوە (ماددە/لاپەڕە). هەڵبژاردە هەڵەکان بە
شێوەی خۆکار لە وەڵامی پرسیارە تردا هەڵدەبژێردرێن.
